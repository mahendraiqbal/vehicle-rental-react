## Installation

1. Clone this repository
```bash
  https://github.com/mahendraiqbal/vehicle-rental-react
```

2. Download all packages

3. Configure .env file 

    REACT_APP_HOST = "https://vehicle-rental-aws.herokuapp.com"

4. Run npm start
