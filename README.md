## Installation

1. Clone this repository
```bash
  $ https://github.com/mahendraiqbal/vehicle-rental-react
```

2. Download all packages
```bash
  $ npm install
```

3. Configure .env file 
```bash
  REACT_APP_HOST = "http://localhost:port"
```

4. Run npm start

```bash
  $ npm start
```
